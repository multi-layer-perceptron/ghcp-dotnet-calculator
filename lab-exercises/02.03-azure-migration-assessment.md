## Exercise 02.03 - Azure Migration Assessment

**Module:** 02 - Modernize And Migrate
**Associated prompt:** [3.02-migrate-to-azure.prompt.md](../.github/prompts/3.02-migrate-to-azure.prompt.md)

### Learning Objectives

* Assess a small application's readiness for Azure migration.
* Produce a phase-based migration baseline plan as a reviewable document.
* Understand which Azure services fit a console-plus-web calculator workload
  with a PostgreSQL test data dependency.
* Practice using Copilot with Azure assessment tooling.

### Overview Of The Prompt

The `3.02` prompt runs an Azure migration readiness assessment against the
calculator workspace and generates a baseline plan. Compare your output with
the committed example in
[docs/azure-migration-assessment-3.02.md](../docs/azure-migration-assessment-3.02.md).

### Steps

1. Complete [Exercise 02.02](02.02-refactor-to-blazor.md), or at minimum
   Module 01, so there is a real workload to assess.
2. Sign in to Azure tooling if the prompt requires it. An Azure subscription
   is optional for the assessment-only portion.
3. In Copilot Chat, run the `3.02` migration prompt.
4. Review the generated assessment and phase plan.

### Success Criteria

* An assessment document identifies workload components, dependencies, and
  candidate Azure services.
* The plan is organized into phases with clear entry and exit criteria.
