## Support Model

Support is community-based and best effort. This repository does not provide
production support, consulting, guaranteed responses, or a service-level agreement.

## Choose The Right Channel

| Need | Channel |
| --- | --- |
| Setup, exercise, or usage question | [GitHub Discussions](https://github.com/multi-layer-perceptron/ghcp-dotnet-calculator/discussions) |
| Idea or broad improvement | [GitHub Discussions](https://github.com/multi-layer-perceptron/ghcp-dotnet-calculator/discussions) |
| Reproducible repository defect | [GitHub Issues](https://github.com/multi-layer-perceptron/ghcp-dotnet-calculator/issues) |
| Documentation or lab error | [GitHub Issues](https://github.com/multi-layer-perceptron/ghcp-dotnet-calculator/issues) |
| Suspected vulnerability | Private process in [SECURITY.md](SECURITY.md) |
| Conduct concern | Private process in [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) |

Search existing content before starting a thread. Include the exercise,
command, expected result, actual result, environment, and non-sensitive evidence
needed to understand the problem.

## Protect Sensitive Information

Do not post passwords, tokens, connection strings, private repository content,
personal data, confidential information, or vulnerability details publicly.
Redact logs and screenshots before sharing them.

## Third-Party Services

Questions about GitHub, GitHub Copilot, Azure, Docker, extensions, packages, or
other third-party services may need the provider's documentation and support
channels. Their terms, policies, availability, and charges apply independently.
